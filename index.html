Judge Business School Charting Tools

Overview

The current list of available charts is as follows:

  • Horizontal Bar Chart
  • Vertical Bar Chart
  • Pie Chart
  • Layered Pie Chart
  • Global Map

All charts use JSON data for configuration, with the exception of the Global Map which uses a CSV file, this is purely for ease of editing.

Required files

The following files are required to be included on all pages with any chart on it:

    /js/d3/d3.min.js
    /js/s24/s24.charts.js
    /js/s24/s24.charts.css

The following file is required if the page has a Global Map on it:

    /js/d3/toppjson.min.js

All files in the /js/d3 folder are required for the Global Map pages (mercator.json and world.json), however these are automatically included in the script.

Setup

Before any charts are initialised on the page, the charting library must be first be initialised:

    S24.Charts.init({
        colourScheme: '/js/s24/colours.json'
    });

The colour scheme is a changeable option, so if two pages have different colours, this can be updated.

Accessibility

All of the charts in the library can be made accessible by adding a title and description tag into the svg element. This is easily configurable and can be done from the standard configuration. For example:

    S24.Charts.createBarChart('.container', '../js/data/data-file.json', {
        title: 'Chart Title',
        description: 'Testing description'
    });

The above options are available on all types of charts across the library.

Editing Colours

The colour scheme for the charts is located in a JSON file located at:

    /js/s24/colours.json

The bar charts and the map will use the "primary" colours in the schema. This is editable by changing the values in this section:

    "primary" : {
        "colour" : "#e80087",
        "font" : "#ffffff"
    },

The pie/donut charts use multiple colours, a different colour for each section of the pie. This means that the configuration looks slightly different:

    "data-green" : {
        "1" : {
            "colour" : "#0C8D78",
            "inactive" : "#EBEBEB"
        },
        "2" : {
            "colour" : "#51B756",
            "inactive" : "#DEDCDD"
        },
        "3" : {
            "colour" : "#ACD59F",
            "inactive" : "#CFCFCF"
        }
    },

These sections can have as many colours as desired. The colours will loop around to the start again if there are not enough for the amount of data values.

The colours that the pie charts will use depend on an option that is passed into the createPieChart (or createLayeredPieChart) method. For example:

    S24.Charts.createLayeredPieChart('.svg-container', '../js/data/mfin-employment-breakdown.json', {
        colours: "data-green",
    });
