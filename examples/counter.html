Counters

Counters are a bit different to most chart types, as they do not require a data file. Instead, they are initiated by specifying the start and end values in the declaration.

The options still work the same way as all other charts and are still added at the end of the declaration.

Counter 1-999 (5000 ms)

Code from this example

Studio24.Charts.createCounter('.svg-container-1', 1, 999, {
    duration: 5000
});

Counter 1000-99999 (10000 ms)

Code from this example

Studio24.Charts.createCounter('.svg-container-2', 1000, 99999, {
    duration: 10000,
    width: 200,
    height: 100,
    color: '#f00',
    fontstyle: 'italic',
    fontsize: '22px'
})
