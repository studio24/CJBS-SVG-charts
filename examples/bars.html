Bar Charts

Bar charts use JSON for the data that they use. An example is the following:

There are two types of bar charts, which can be invoked in the following ways:

Code from this example

Studio24.Charts.createHorizontalBarChart('.svg-container-1', '../js/data/mfin-employment-breakdown.json', {
    legendWidth: 200,
    width : 600,
    height: 300
});

Data from this example

[
    { "key" : "New job", "value" : "21" },
    { "key" : "Returned to Employer", "value" : "14" },
    { "key" : "Internship", "value" : "3" },
    { "key" : "Started own Company", "value" : "3" }
]
