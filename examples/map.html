Maps

Maps are the only type of chart which are invoked with CSV data. This is because a CSV file is a more efficient way of storing data with more columns.

Post MFin Location

Code from this example

Studio24.Charts.createMap('.svg-container-1', '../js/data/mfin-post-mfin-location.csv', {
    worldJson: '../js/d3/world.json',
    width: 800,
    legendWidth: 100
});
