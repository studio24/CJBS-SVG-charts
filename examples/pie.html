Pie Charts

The data for pie charts is stored in JSON, in exactly the same way as bar charts (see above).

Pie charts are called in much the same way as Bar Charts. Again, there are two different types: pie chart and layered pie chart.

Career Outcome Overview

Code from this example

Studio24.Charts.createPieChart('.svg-container-1', '../js/data/mfin-career-overview.json', {
    title: "Career Outcome Overview",
    colours: "data-pink",
    width: 350,
    height: 300
});

Data from this example

[
    { "key" : "Employed", "value" : "41" },
    { "key" : "Still looking", "value" : "3" }
]

Breakdown of those Employed
